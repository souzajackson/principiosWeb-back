# PetConnect - Principios de Dev Web

Esse projeto se trata de uma aplicação web para auxiliar e facilitar todo o processo de adoção de animais. Além disso a aplicação quer ajudar os abrigos a arrecadar doações e, se possível, agendar visitas

## Entidades
- Usuário
- Animal
- Abrigo
- Adoção
- Doação

## Funcionalidades
- Usuário se cadastrar
- Responsável por abrigo se cadastrar
- Usuário adotar animal
- Usuário agendar visita a abrigo
- Abrigo cadastrar animal
- Usuário doar para abrigo
- Abrigo confirma adoação de animal
